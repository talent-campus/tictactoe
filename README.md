Tic Tac Toe in Python
=====================

A few simple, text-based Tic Tac Toe game implementations. To help you learn Python.

Instructions
------------

Run the programs using Python, e.g.
```
python3 tictactoe_humans.py
```
```
python3 tictactoe_computer.py
```
